Mangaku

Mangaku es una página web dedicada a la compra y venta de manga en formato físico. Este proyecto fue creado para una asignatura de la univeridad. Desarrollé este proyecto desde cero utilizando HTML, CSS y JavaScript para el frontend, y PHP junto con MySQL para el backend.

La web cuenta con un apartado donde se pueden ver todos los mangas y una página plantilla para visualizar los detalles de uno de ellos.

Otro apartado de la página, es un blog donde se almacenan mensajes anonimos de los usuarios, esto para añadir contenido a la base de datos.
